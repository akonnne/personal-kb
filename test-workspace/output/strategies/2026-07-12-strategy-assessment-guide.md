# 澳洲职业评估实战策略（2026版）

## 背景

职业评估是澳洲技术移民的"入场券"——没有有效的职业评估，EOI 连递交资格都没有。本文基于 2026 年最新政策，对比分析 ACS（IT类）、EA（工程类）、VETASSESS（通用类）三大评估机构的核心要求、费用差异和避坑要点。

## 核心结论

**职业评估不选错机构，比考雅思重要十倍。** 同一个职业可能存在多个可选评估机构（如软件工程师可选 ACS 或 EA），选错机构可能导致评估失败、CDR 抄袭被禁申 12 个月，或者严重延误移民时间线。关键原则：先确认 ANZSCO 职业代码 → 查评估机构官网确认是否覆盖 → 对比费用和周期 → 选定后不轻易换。

## 三大评估机构对比

| 维度 | ACS（IT类） | EA（工程类） | VETASSESS（通用类） |
|------|-----------|------------|-------------------|
| 覆盖职业 | 软件工程师、网络工程师、数据科学家等 50+ | 土木、机械、电气、化学、软件等 30+ | 覆盖数百个职业（组A-F） |
| 评估费用 | $560（标准） | A$850-1,200 | A$800-1,200 |
| 审理周期 | 8-12周（加急2周） | CDR 8-12周 / 认可学历 4-8周 | 12-16周（加急6-8周） |
| 核心材料 | 雇主推荐信+薪资证明 | CDR报告（非认可学历） | 学历认证+工作证明+收入证明 |
| 有效期 | **2年**（2026年新规） | 3年 | 2年 |
| PY加分 | ✅ IT PY | ✅ 工程PY | ❌ |
| 最大风险 | 非ICT学历需6-8年经验 | CDR抄袭被禁12个月 | 材料不全直接拒评（Decision-Ready新政） |

## ACS 评估详解

**2026 关键变化**：评估报告有效期从 3 年缩短至 **2 年**，过期需重新评估（费用 $605）。非相关学历的工作年限抵扣大幅收紧。

### 评估类型选择

| 类型 | 适用人群 | 工作经验要求 | 费用 |
|------|---------|------------|------|
| 澳洲毕业生类 | 澳洲 ICT 学历 + PY | 1 年或 PY 替代 | $560 |
| 海外学历类 | ICT 主修（海外） | **2年**相关经验 | $560 |
| 海外学历类 | ICT 辅修（海外） | **5年**相关经验 | $560 |
| RPL 类 | 非 ICT 学历 | **6-8年**经验 + 2份项目报告 | $905 |
| 临时毕业生类 | 申请 485 工签 | 无需 | $560 |

### 材料清单
1. **护照** — 首页彩色扫描
2. **学历** — 学位证、毕业证、成绩单（中英文公证）+ 2025年新政要求课程大纲
3. **雇主推荐信** — 公司抬头纸 + 详细职责描述（匹配ANZSCO）+ 起止日期 + 签字
4. **收入证明** — 工资单、税单、社保（至少2项）
5. **RPL 额外** — 2 份项目报告（近2年和近4年内各1份）

### ACS 时间线
```
准备材料（1-2月）→ 提交申请 → 等待审理（8-12周/加急2周）→ 出结果
建议：提前 3-4 个月准备，与雅思备考并行
```

## EA 认证详解

### CDR — 最大拦路虎

非认可学历（如中国大部分工程学位）必须提交 CDR 报告，包含：
- **3 篇 Career Episode**（每篇 1000-2500 字），展示 16 项核心工程能力
- **Summary Statement** — 将 CE 内容映射到 EA 能力标准
- **CPD** — 持续学习发展记录

**⚠️ 致命风险**：EA 已启用高级查重系统。CDR 抄袭 → 拒签 + **禁申 12 个月**。中介代写同样可能触雷。

### 费用明细

| 项目 | 费用 |
|------|------|
| 标准评估 | A$850-1,200 |
| 加急（6-8周） | +A$300-500 |
| 复审 | A$400-600 |
| CDR 撰写服务 | A$1,500-3,000（专业机构） |

### 工程师移民凑分公式
```
年龄 25-32:     30分
英语 4个8:       20分
澳洲学历:        5分
偏远地区学习:    5分
PY职业年:        5分
NAATI/CCL:       5分
工作经验 3-5年:  5分
配偶加分:       10分
────────────────────
Total: 70-85分
```

## VETASSESS 详解

**2026 Decision-Ready 新政**：材料不全 **直接拒评**，不再给补件窗口。这是最容易被忽视的风险点。

### 职业分组与要求

| 组别 | 最低学历 | 工作经验 |
|------|---------|---------|
| A | 学士+相关专业 | 5年相关（其中至少1年近5年） |
| B | 学士+非相关 | 3年+专科相关学历 |
| C | 专科+相关 | 2年 |
| D | 专科+非相关 | 3年 |
| E | 高中 | 5年 |
| F | 高中 | 1年（极少职业） |

### ⚠️ 中国学历必须 CQV 认证

VETASSESS 要求所有中国学历先通过 **CQV（中国学历认证）**，这是多出来的 2-3 个月时间成本。建议：**确认要申请技术移民的那一刻，就启动 CQV**。

## 职业评估常见失败原因

| 原因 | 占比 | 主要涉及机构 | 补救方案 |
|------|------|------------|---------|
| 学历资质不匹配 | 35% | ACS/VETASSESS | 补相关课程/积累更多工作经验 |
| 工作证明材料不足 | 25% | 全部机构 | 完善雇主信（含具体职责+时长） |
| CDR/报告抄袭 | 15% | EA | 重写 CDR，6 个月后重新提交 |
| 选错评估机构 | 10% | 全部 | 退款后重新选择，不可转移 |
| 收入证明不充分 | 15% | VETASSESS | 补充个税+银行流水+养老金 |

## 行动建议

1. **提前 6 个月启动** — 职业评估不是"填表提交就行"，CDR 撰写耗时 1-3 个月，CQV 认证需 2-3 个月
2. **ACS 申请人** — 注意有效期缩至 2 年，计划好时间别过期
3. **EA CDR 路径** — 严禁抄袭。CDR 质不足可找专业服务，但确保内容基于真实经历
4. **VETASSESS 申请人** — 先做 CQV，材料准备完整再提交（Decision-Ready 不给补件机会）
5. **并行推进** — 职业评估和雅思备考并行，不要串行等
