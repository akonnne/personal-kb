# 澳洲ACS职业评估具体申请步骤和材料，2026年最新

澳洲ACS职业评估申请需完成6个步骤：确认提名职业→创建ACS在线账户→准备并上传材料→支付评估费用→等待审理（8-12周）→接收结果。核心材料包括护照、学历学位证及成绩单、雇主推荐信（含详细职责描述）、薪资证明。RPL路径需额外提交两份项目报告。费用为$560-$905澳元，结果有效期2年。

## 一、ACS职业评估是什么

1. ACS：Australian Computer Society，澳洲计算机协会
2. 用途：IT类职业申请澳洲技术移民的必经之路
3. 覆盖职业：软件工程师、网络工程师、数据分析师等50+职业
4. 有效期：2年

## 二、申请步骤详解

1. 步骤1：确认提名职业与评估类型
2. 步骤2：创建ACS在线账户（acs.org.au）
3. 步骤3：在线填写申请表（个人信息、学历、工作经验）
4. 步骤4：上传材料（PDF格式，每个文件不超过3MB）
5. 步骤5：支付费用（信用卡或银行转账）
6. 步骤6：等待审理（标准8-12周，加急2周）

## 三、所需材料清单

1. 身份证明：护照首页彩色扫描件
2. 学历材料：学位证、毕业证、成绩单（中英文公证）
3. 课程大纲：2025年新政要求，境外学历需提供
4. 雇主推荐信：公司抬头纸、职位、职责、起止日期、推荐人签字
5. 收入证明：工资单、税单、社保记录（至少2项）
6. RPL额外材料：2份项目报告（近2年和近4年内各1份）

## 四、评估类型

1. 澳洲毕业生类：澳洲ICT学历+PY职业年或1年工作经验
2. 海外学历类：ICT主修需2年经验，辅修需5年经验
3. RPL类：非ICT学历需6-8年经验+2份项目报告
4. 临时毕业生类：申请485工签，无需工作经验

## 五、费用明细

1. 标准评估费：$560澳元
2. 加急费：额外$345澳元（2周出结果）
3. 复审费：$395澳元
4. 重新评估费：$605澳元
5. 额外职业评估：每个额外$145澳元

## 六、注意事项

1. 提前3-4个月准备，避免延误移民申请
2. 雇主推荐信职责描述要详细，匹配ANZSCO要求
3. 中国学历需做清华认证或学信网认证
4. 2025年新政要求提供课程大纲
